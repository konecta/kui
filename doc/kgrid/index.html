kGrid

Grilla para aplicaciones web de kUI.

El proyecto kGrid es una librería javascript que responde a las necesidades más comunes de listado, edición y selección. Busca ser un reemplazo para Jqgrid en los proyectos desarrollados con Bootstrap, en consecuencia el formato requerido para los servicios es el mismo que el de dicho plugin.

Se centra en las opciones de edición rápida y las tarjetas de información. En esta etapa de desarrollo ya cuenta con soporte parcial para responsive design, especialmente en el tipo grilla.

Funciones

Formato de funciones: $('#mi-div').kGrid('nombre de la función','objeto parámetro')

Función Descripción Formato del parámetro
recargar Recargar la grilla haciendo una petición AJAX con los parametros de data. El parámetro reemplaza la configuración del data . Es opcional y en caso de omisión se mantiene la configuración previa.
pagina Cargar a la grilla los datos de la página especificada. El parámetro es requerido y puede tener uno de los siguientes valores:
  • "primera"
  • "anterior"
  • "siguiente"
  • "ultima"
  • El número de la página específica que desea cargarse.
buscar Filtrar los resultados bajo cierto criterio o clave de búsqueda. 
{
    data: 'clave de busqueda global (requerido)',
    reglas: [
             field: 'nombre del campo (requerido)',
             data: 'opcional, por defecto toma el valor de aux.data',
             op: 'opcional, por defecto cn'
         ]
    groupOp: 'opcional, por defecto es AND'
}
seleccionar Seleccionar todas las entradas indicadas en el parámetro, sólo se puede utilizar cuando está habilitada la opción seleccionable. Array con los ids de las entradas a seleccionar.
agregar Agregar nueva entrada a la grilla. Objeto que se agregará al final de la página actual, en caso de omisión se agregará una entrada vacía editable al principio de dicha página.

Data

Información accesible con jQuery: $('#mi-div').data('nombre del dato')

Dato Descripción
datos Retorna un objeto donde los objetos recibidos en la página actual se mapean a sus respectivos identificadores únicos o Primary Keys.
totalDatos Cantidad total de entradas.
pagina Página actual
totalPaginas Cantidad total de páginas.
seleccionados Retorna el array de ids de las entradas seleccionadas. Solo funciona en el formato simple y cuando seleccionable=true.

Personalización


Temas

Para personalizar la apariencia de kGrid se cuenta con soporte para temas. Para aplicar un tema en particular sólo se debe importar una hoja de estilos (css) en el <head>.


Crear tu propio tema

Para crear tu propio tema solo debes reescribir el código del tema por defecto.

.kgrid .kbtn {
	cursor:pointer;
}

.kgrid .kbtn:hover {
	background:#ECECF0;
	border:1px solid #cacaca;
}

.kgrid h2 {
	padding-bottom:10px;
}

.kgrid .kscore {
	width:0;
	height:0;
	border:1px solid #999;
	background:#ECECF0;
	font-size:1.7em;
	margin-left:10px;
	overflow:hidden;
	border-radius:50%;
}

.kgrid .kscore p {
	margin-top:25%;
}

.kgrid .kscore small {
	font-size:.5em;
}

.kgrid .klabel {
	margin-top:20px;
}

.kgrid .kaccion {
	margin-left:15px;
}

.kgrid .kpagina {
	overflow:hidden;
	padding-top:2px;
}

.kgrid .kpagina input {
	text-align:center;
	margin:0 5px;
}
